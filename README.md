How to Play (Korean)
=======================
- 룸1에서는 룸2로 가기 위해서는 화분을 치워서 열쇠를 주워야 함

- 룸2에서 숨겨진 문을 찾기 위해서 화분을 치워야 함

- 룸2에서 숨겨진 문을 열기 위해서는 룸2의 불을 꺼서 암호를 확인후 password를 입력해야함
